Architecture Overview

This site demonstrates the complete CSS architecture in action:

  • Tokens: Shared design tokens (colors, spacing, typography, shadows) defined across the site
  • Base: Element resets and semantic styling
  • Layout: Page structure and responsive grid systems
  • Components: Reusable UI components (cards, navigation)
  • Utilities: Helper classes for spacing and text styling

Built with CSS Layers, Lightning CSS, and responsive design principles
